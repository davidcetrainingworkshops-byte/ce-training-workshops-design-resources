CE Training Lesson 1932

Install:
1. Upload ce-training-lesson-1932-plugin.zip in WordPress under Plugins > Add New > Upload Plugin.
2. Activate "CE Training Lesson 1932".
3. Place this shortcode in the lesson page:

[ce_lesson_1932]

Alternate shortcode:

[ce_training_lesson_1932]

This plugin renders the CE Training Workshops video player, practice quiz, and chat replay from PHP and enqueues the CSS/JS as real WordPress assets. The chat messages are embedded in the template so they are not dependent on Enfold preserving large inline editor code.
