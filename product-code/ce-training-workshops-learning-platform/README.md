# CE Training Workshops Learning Platform

This folder contains representative frontend-focused WordPress and Sensei LMS code for the broader CE Training Workshops learning platform. It demonstrates the branded video, practice-activity, and supporting course-interaction experience.

## Install

1. Package this folder as a ZIP.
2. Upload it in WordPress under **Plugins > Add New > Upload Plugin**.
3. Activate **CE Training Workshops Learning Platform**.
4. Place this shortcode on an appropriate course or learning-experience page:

`[ce_training_experience]`

Alternate shortcode: `[ce_training_platform]`

The code enqueues CSS and JavaScript as real WordPress assets and renders the learning experience from a reusable PHP template.
